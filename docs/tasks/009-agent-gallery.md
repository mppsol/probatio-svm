# Task 009 — Live-agent gallery (certification transcripts)

**Owner:** CC (frame-thin: tooling / serialization).
**Reviewer:** Codex.
**Branch:** `task/009-agent-gallery`.
**Depends on:** Task 008 merged.
**Motivation:** pitch evidence. Run a real Claude agent through episodes under a mandate and save a
**transcript** of what it did + Probatio's certification verdict. Turns "we can certify agents" from a
claim into an artifact. The live path needs `ANTHROPIC_API_KEY` and is non-deterministic, so this is a
subcommand + saved files, NOT a test — but the transcript builder is pure and offline-tested, and a
deterministic **scripted sample** is committed so the format is visible without a key.

## Design

- **`crates/harness/src/transcript.rs`** — pure, offline-testable:
  - `SlotRecord { slot, mark, measured_delta, aggregate_delta, any_liquidatable }`.
  - `Transcript { label, system, claimed_delta, claims_solvent, backend, verdict, findings:
    Vec<(kind, evidence_slots)>, slots: Vec<SlotRecord> }`.
  - `Transcript::capture(label: &str, backend: &str, ep: &EpisodeResult, report: &ShortcutReport) -> Self`
    (reads the trace + claim + report — all already available).
  - `to_json(&self) -> String` via `serde_json::json!` (serde_json is already a dep; no serde derive).
- **`main.rs` `gallery` subcommand:**
  - `gallery --sample` → deterministic: run a `ScriptedDecider` drift policy under `NEUTRAL_MM` on the
    clean ref episode, write `gallery/sample-scripted-drift.json`, print a one-line summary. **No key
    needed.** This is the committed, format-accurate illustration.
  - `gallery [--hostile]` → live: run `ClaudeAgent{NEUTRAL_MM}` via `CurlClaude` on the clean (or
    hostile) ref episode, write `gallery/neutral_mm-<clean|hostile>.json`, print verdict + findings.
    Requires the key; graceful degrade (missing key ⇒ message + exit 1, same as `agent`).
- **`gallery/README.md`** — explains the format, that `sample-*.json` is a scripted illustration (not a
  live Claude run), and how to generate real transcripts (`cargo run -- gallery` with the key set).
- Commit `gallery/sample-scripted-drift.json` (generated by `gallery --sample`).

## Acceptance criteria

- Offline: a test builds a `Transcript` from a `ScriptedDecider` drift run + `verify`, and asserts the
  JSON contains the mandate label, `verdict == "ShortcutDetected"`, a `ClaimTracksExposure` finding, and
  `slots.len() == N_SLOTS`.
- `cargo run --offline -p probatio-svm-harness -- gallery --sample` writes
  `gallery/sample-scripted-drift.json` deterministically (same bytes each run) and exits 0 without a key.
- `gallery` (no `--sample`, no key) prints the key-required message and exits 1.
- All prior 49 tests still green; `cargo test --offline` green; no warnings. The committed sample file
  matches what `--sample` produces.

## Out of scope

- Live transcripts committed to the repo (Hiro generates those with a key — the sample suffices for the
  format). CPI guard promotion, pitch video.
